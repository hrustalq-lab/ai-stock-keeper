# AI Stock Keeper

AI-powered stock management system for 1C ERP integration. Automates goods intake, inventory control, and stock notifications.

## Features

- **AI-Powered Intake**: Intelligently pick and fulfill goods into 1C accounting system
- **Smart Notifications**: Real-time inventory alerts and stock level monitoring
- **1C Integration**: Seamless integration with Russian 1C ERP platform
- **Stock Control**: Automated tracking and management of warehouse inventory

## Tech Stack

This is a [T3 Stack](https://create.t3.gg/) project with:

- [Next.js](https://nextjs.org) - React framework
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [tRPC](https://trpc.io) - Type-safe API routes
- [ESLint](https://eslint.org/) & [Prettier](https://prettier.io/) - Code quality
